# Video Transcriber

Простой инструмент для автоматической транскрипции аудио и видео файлов. Работает локально на вашем компьютере, не требует GPU.

## Возможности

- Транскрипция локальных файлов (mp4, mp3, wav, m4a и другие)
- Скачивание и транскрипция с YouTube
- Поддержка русского и английского языков
- Работает на обычном процессоре (CPU)
- Показывает прогресс в реальном времени
- Автоматически фильтрует тишину и шумы

## Установка

### Требования

- Python 3.10 или новее
- FFmpeg (для обработки видео/аудио файлов)

### Быстрая установка

**Windows:**

cmd

```
install.bat
```

**Linux/Mac:**

Bash

```
chmod +x install.sh
./install.sh
```

### Установка вручную

1. Скачайте проект:

Bash

```
git clone <ссылка-на-репозиторий>
cd video-transcriber
```

2. Создайте виртуальное окружение:

Bash

```
python -m venv .venv

# Windows:
.venv\Scripts\activate

# Linux/Mac:
source .venv/bin/activate
```

3. Установите зависимости:

Bash

```
pip install -r requirements.txt
```

### Установка FFmpeg

FFmpeg необходим для работы с видео и аудио файлами.

**Windows:**

cmd

```
winget install ffmpeg
```

**Ubuntu/Debian:**

Bash

```
sudo apt install ffmpeg
```

**macOS:**

Bash

```
brew install ffmpeg
```

Проверьте установку:

Bash

```
ffmpeg -version
```

## Использование

### Базовые примеры

Транскрибировать локальный файл:

Bash

```
python transcriber.py -i "video.mp4"
```

YouTube видео:

Bash

```
python transcriber.py -i "https://youtube.com/watch?v=..."
```

Английское аудио:

Bash

```
python transcriber.py -i "podcast.mp3" -lang en
```

### Выбор модели

Доступно несколько моделей с разным балансом скорости и качества:

Bash

```
# Быстрая обработка (рекомендуется для первого раза)
python transcriber.py -i "file.mp4" -m turbo

# Баланс скорости и качества (по умолчанию)
python transcriber.py -i "file.mp4" -m large-v3

# Для более коротких файлов
python transcriber.py -i "file.mp4" -m medium
```

## Сравнение моделей

|Модель|Скорость|Качество|Языки|Рекомендация|
|---|---|---|---|---|
|`turbo`|Очень быстро|Хорошее|Все|Для большинства задач|
|`large-v3`|Быстро|Отличное|Все|Когда нужно качество|
|`large`|Средне|Отличное|Все|Альтернатива v3|
|`medium`|Быстро|Хорошее|Все|Для коротких файлов|
|`small`|Очень быстро|Среднее|Все|Для черновиков|

## Параметры

|Параметр|Описание|Значение по умолчанию|
|---|---|---|
|`-i, --input`|Путь к файлу или URL|Обязательный|
|`-lang, --language`|Язык транскрипции|`ru`|
|`-m, --model`|Размер модели|`large-v3`|

## Результаты

Все транскрипции сохраняются в папку `transcripts/` в формате SubRip (.srt) с временными метками.

Пример структуры:

text

```
transcripts/
├── interview_2024.srt
├── podcast_episode_5.srt
└── YouTube_Video_Title.srt
```

Файл .srt можно открыть в текстовом редакторе или использовать как субтитры к видео.

## Производительность

Примерное время обработки 15-минутного аудио на обычном процессоре:

- **turbo**: ~6-8 минут
- **large-v3**: ~8-12 минут
- **medium**: ~5-7 минут

Первый запуск займет больше времени из-за скачивания модели (~1-3 ГБ в зависимости от выбранной).

## Системные требования

**Минимальные:**

- RAM: 4 ГБ
- Свободное место: 3 ГБ (для моделей и кэша)
- Процессор: любой современный CPU

**Рекомендуемые:**

- RAM: 8 ГБ или больше
- SSD диск (заметно ускоряет работу)
- Многоядерный процессор

## Решение проблем

### FFmpeg не найден

Bash

```
pip install ffmpeg-python
```

Также убедитесь, что FFmpeg установлен в системе (см. раздел "Установка FFmpeg").

### Ошибка при скачивании YouTube

Bash

```
pip install --upgrade yt-dlp
```

### Медленная работа

- Используйте модель `turbo` или `medium`
- Убедитесь, что у вас достаточно оперативной памяти
- Закройте другие программы во время транскрипции

### Плохое качество распознавания

- Проверьте качество исходного аудио
- Используйте модель `large-v3` для лучшего результата
- Убедитесь, что правильно указан язык (`-lang ru` или `-lang en`)

### Первый запуск очень долгий

Это нормально - программа скачивает модель (от 600 МБ до 3 ГБ). Последующие запуски будут быстрее.

## Техническая информация

Программа использует:

- **faster-whisper** - оптимизированная реализация Whisper на основе CTranslate2
- **int8 квантизация** - для уменьшения использования памяти
- **VAD (Voice Activity Detection)** - автоматическая фильтрация тишины и шумов
- **Многопоточность** - использует все доступные ядра процессора

## Частые вопросы

**Можно ли использовать без интернета?**  
Да, после первого запуска все модели сохраняются локально.

**Поддерживаются ли другие языки кроме русского и английского?**  
Да, модели поддерживают более 50 языков. Укажите код языка в параметре `-lang`.

**Есть ли ограничения по длительности файла?**  
Нет, можно обрабатывать файлы любой длительности.

**Почему результат может отличаться от оригинала?**  
Качество зависит от четкости речи, фонового шума и выбранной модели. Для критичных задач используйте `large-v3`.
